## How It Works

### Path Patterns in .gitignore

The paths in subdirectory `.gitignore` files use **relative paths** (starting with `/`):

- `/dist` in `webpage/.gitignore` → ignores `webpage/dist/` ✅
- `/node_modules` in `webpage/.gitignore` → ignores `webpage/node_modules/` ✅
- `/build/` in `flutter_app/.gitignore` → ignores `flutter_app/build/` ✅

This is **correct behavior** - the `/` prefix means "relative to this directory".
